# 微生物名稱拉丁文拼字大賽

## 發音

- -ae, -oe, -eae: /e/
- th: /t/
- c + e, i, y, ae, oe: /s/
- c + 其他母音: /k/
- ch: /k/
- rh: 打舌的r（崩潰）
- ti + 母音: /s/ + i + 母音
- s/t/x/ + tia: /t/ + i + 母音
